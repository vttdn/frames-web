功能特色: 類比攝影應用功能

歸檔

整理圖庫

輕鬆瀏覽、整理和管理所有膠卷，保持每次拍攝會話結構清晰、易於查找且完美有序。

設備

管理設備

保留相機、鏡頭和濾鏡的詳細清單，以保持一致性並為每次拍攝準備好所有設備資訊。

記錄

捕捉設定

直接記錄每張照片的光圈、快門速度、膠片類型和位置詳情，保持每一格的資訊準確且可存取。

歷史

回顧照片

回顧已記錄的膠卷以查看設定，分析之前的拍攝會話，從中學習並改進您的膠片攝影工作流程。

導出

導出記錄

將記錄保存或分享為 .frames、.txt 或 .gpx 檔案，用於備份、分享或與其他應用或工作流程整合。

元資料

保存元資料

將所有記錄的資料嵌入掃描影像以保留膠片資訊，同時保持數位檔案完全同步。

支援

電子郵件支援

隨時直接聯絡開發者尋求協助、分享回饋或建議新功能和改進，並獲得快速回覆。

常見問題: 來自膠片攝影師

我可以記錄哪些元資料？

Frames 允許您為每張照片記錄詳細資訊，包括相機、鏡頭、濾鏡、畫幅編號、焦距、光圈、快門速度、ISO、測光模式、膠片類型、閃光燈、GPS 位置、時間戳記和自訂筆記，確保您永遠不會遺失重要的膠片攝影細節。

支援哪些影像格式？

Frames 支援 JPEG、TIFF 和 DNG 格式，允許您將記錄的元資料直接嵌入掃描的膠片照片中，以便在所有裝置之間無縫組織和工作流程。

Frames 有 Android 或 Windows 版本嗎？

Frames 目前適用於 iPhone 和 macOS。Android 和 Windows 版本計畫於 2026 年推出，在此期間，您可以透過 .frames 檔案導出分享和管理您的資料。

支援哪些快門速度和光圈？

Frames 提供涵蓋大多數鏡頭和相機的預設序列，您還可以為您的設備定義自訂序列，使其與任何快門速度或光圈值相容。

我可以記錄三腳架或配件嗎？

是的，Frames 允許您記錄三腳架、閃光燈和其他配件。還有一個額外的筆記欄位，您可以在其中添加任何想要的自訂細節。

未來更新有什麼計畫？

未來計畫包括擴充的工作流程功能、改進的跨設備整合以及社群要求的新工具，使膠片攝影記錄更加靈活和強大。
