개인정보 보호정책

Frames는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 iOS 및 macOS 앱과 웹사이트 전반에서 정보가 어떻게 처리되는지를 설명합니다. 우리는 데이터 보안을 유지하고, 투명하게 관리하며, 사용자가 자신의 정보를 완전히 통제할 수 있도록 하는 데 전념하고 있습니다. 불필요한 수집이나 추적은 전혀 이루어지지 않으며, 공유 또는 내보내기 여부는 항상 사용자의 선택에 달려 있습니다.

iOS 및 macOS용 Frames

Frames는 개인 데이터를 수집, 저장 또는 공유하지 않습니다. 앱에 입력한 모든 정보는 사용자의 기기에만 로컬로 저장되며, 서버나 제3자에게 전송되지 않습니다. 앱은 Apple의 StoreKit 프레임워크를 사용하여 구독을 관리합니다. 이 과정은 전적으로 Apple에 의해 처리되며, 개발자는 어떠한 개인 정보에도 접근할 수 없습니다.

Frames에는 분석 도구, 추적 기능 또는 온라인 서비스가 포함되어 있지 않습니다. 네트워크 연결은 사용자가 선택적으로 위치 정보를 기록할 때 iOS의 CLLocationManager를 통해 위치 서비스를 사용할 때에만 발생합니다.

데이터를 내보낼 때 앱은 .frames, .txt 또는 .gpx 파일을 생성할 수 있습니다. 이 파일들은 시스템 공유 메뉴를 통해 공유할 수 있으며, 모든 공유 동작은 전적으로 사용자의 제어하에 있습니다. 개발자는 내보낸 콘텐츠에 접근하거나 이를 추적하지 않습니다. Frames는 다음과 같은 시스템 권한을 요청합니다:

  • 카메라: 앱 내에서 직접 사진을 촬영하기 위해 사용됩니다.
  • 사진: 사진 앱에서 이미지를 선택하기 위해 사용됩니다.
  • 위치 정보 (앱 사용 중): 프레임을 기록할 때 GPS 좌표를 저장하기 위해 사용됩니다.

withframes.com 웹사이트

Frames 웹사이트는 GitHub Pages에서 호스팅되는 정적 사이트입니다.
쿠키, 분석 도구 또는 추적 스크립트를 사용하지 않습니다. 어떠한 개인 데이터도 수집하거나 저장하지 않습니다. Cloudflare는 서버 로그를 기반으로 익명 트래픽 통계를 제공합니다. 이러한 로그에는 개인 식별 정보가 포함되어 있지 않으며, 프로파일링이나 마케팅 목적으로 사용되지 않습니다.

웹사이트에는 콘텐츠 공유나 Frames가 소개된 외부 웹사이트 및 소셜 미디어 링크가 포함될 수 있습니다. 이러한 사이트 방문은 선택 사항이며, 각 사이트의 개인정보처리방침이 적용됩니다. 이 웹사이트는 양식, 뉴스레터, 또는 기타 상호작용 기능을 통해 정보를 수집하지 않습니다.
