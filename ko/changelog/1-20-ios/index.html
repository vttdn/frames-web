변경 내역다운로드

2026년 01월 15일

필름 정렬 및 빠른 접근

이번 업데이트로 라이브러리 구성을 더 세밀하게 제어하고 탐색 속도가 빨라졌습니다. 폴더 내 필름에 정렬 및 순서 옵션이 추가되었으며, 새로운 최근 필름 바로가기로 현재 화면을 벗어나지 않고 프로젝트 간을 이동할 수 있습니다. 또한 모든 행을 길게 눌러 일반적인 작업에 접근할 수 있으며, 노출 설정에 새로운 자동 옵션이 추가되었습니다

필름 정렬 및 정리

이제 폴더 내 필름 정리를 완벽하게 제어할 수 있습니다. 롤을 이름, 생성일 또는 수정일로 정렬하고 오름차순 또는 내림차순을 선택하세요. 프로젝트, 여행 또는 카메라별로 필름을 그룹화하면 모든 것을 스크롤하지 않고도 원하는 것을 훨씬 쉽게 찾을 수 있습니다

최근 필름으로 이동

새로운 최근 필름 바로가기가 모든 필름 메뉴에서 사용 가능합니다. 라이브러리로 돌아가지 않고도 최근에 작업하던 다른 롤로 빠르게 전환할 수 있습니다. 작은 추가 기능이지만 활성 프로젝트 간을 이동할 때 시간을 절약해 줍니다

길게 눌러 빠른 작업

이제 모든 행을 길게 누르면 해당 항목의 바로가기 메뉴가 표시됩니다. 이동, 편집, 삭제 등 필요한 옵션이 바로 거기에 있습니다. 앱 전체에서 작동하며 작업을 완료하는 데 필요한 탭 수를 줄여줍니다

노출 설정 자동 옵션

프레임을 기록할 때 셔터 속도와 조리개에 자동을 선택할 수 있습니다. 카메라가 노출을 자동으로 처리하고 특정 값을 입력하지 않고 촬영만 기록하고 싶을 때 유용합니다. 이 옵션은 레코더 도구 모음의 기존 수동 선택 옆에 있습니다

위치 및 개인정보 보호 업데이트

위치 서비스를 꺼둔 경우에도 제한 없이 수동으로 위치를 설정할 수 있습니다. 앱에서 위치를 선택하기 위해 더 이상 GPS 접근이 필요하지 않습니다. 또한 데이터 처리 방식을 더 명확하게 보여주는 재설계된 개인정보 보호 설정 화면과 개인정보 보호정책에 대한 직접 접근도 추가되었습니다

개선 사항 (10)

  • 폴더 내 새로운 정렬 및 순서 옵션
  • 모든 필름 메뉴에서 최근 필름 바로가기 사용 가능
  • 모든 행 길게 눌러 상황별 작업에 접근
  • 레코더에서 셔터 속도 및 조리개 새 자동 옵션
  • 위치 서비스 비활성화 시 수동으로 위치 설정
  • 개인정보 보호정책 직접 접근이 포함된 재설계된 개인정보 보호 설정 화면
  • 인터페이스 및 애니메이션 개선
  • 지원되는 모든 언어의 번역 업데이트
  • 형식을 쉽게 찾을 수 있도록 내보내기 메뉴에 아이콘 추가
  • 카메라 권한 설명을 기기 언어에 맞게 현지화

수정 사항 (2)

  • 필름 생성: 새 앱 설치 시 필름을 추가할 수 없는 버그를 수정했습니다
  • 프레임 선택기 순서: 폼 선택기에서 프레임 번호가 순서대로 표시되지 않는 문제를 수정했습니다.

패치 (3)

  • 1.20.1: 새 설치에서 필름 추가 수정
  • 1.20.2: 자동 노출 옵션, 개인정보 보호정책 접근, UI 개선
  • 1.20.3: 더 명확한 내보내기 옵션, 선택기 순서 수정 및 현지화 개선
