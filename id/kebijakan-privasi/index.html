Kebijakan Privasi

Komitmen Kami terhadap Privasi Anda

Aplikasi ini tidak mengumpulkan, menyimpan, atau membagikan data pribadi apa pun. Semua data pengguna disimpan secara lokal di perangkat Anda dan tidak pernah dikirimkan ke server atau pihak ketiga mana pun. Kami tidak menggunakan analitik, pelacakan, atau layanan online yang mengumpulkan informasi pengguna.

Namun, aplikasi memungkinkan Anda untuk mengekspor file. Data apa pun yang Anda pilih untuk dibagikan atau disimpan secara eksternal berada di bawah kendali Anda. Developer tidak melacak atau mengakses file yang diekspor.
