บันทึกการเปลี่ยนแปลง

iOS 1.103 มิ.ย. 2025·บันทึกการเปลี่ยนแปลง

ขอแนะนำ Frames Pro

Frames Pro มาถึงแล้ว นำเสนอม้วนฟิล์มไม่จำกัดและการเข้าถึงฟีเจอร์ขั้นสูงเมื่อเปิดตัว ผู้ใช้ฟรีสามารถใช้ Frames ต่อได้สูงสุด 10 ม้วน ขณะที่ Pro ปลดล็อกศักยภาพเต็มรูปแบบของระบบการทำงานการถ่ายภาพ analog ของคุณ

ปลดล็อกการบันทึกฟิล์มไม่จำกัด

ด้วย Frames Pro คุณสามารถสร้างม้วนฟิล์มไม่จำกัดเพื่อบันทึกคอลเลกชันการถ่ายภาพ analog ทั้งหมดของคุณโดยไม่มีข้อจำกัด ไม่ว่าคุณจะกำลังบันทึกม้วนฟิล์มหลายร้อยม้วนจากการถ่ายมาหลายปีหรือกำลังบันทึกโปรเจกต์ที่กำลังดำเนินการหลายสิบโปรเจกต์อย่างแข็งขัน Frames Pro ขยายตามความต้องการของคุณ ผู้ใช้ฟรียังคงเข้าถึงฟีเจอร์หลักทั้งหมดได้เต็มรูปแบบด้วยขอบเขต 10 ม้วน ทำให้ Frames เข้าถึงได้สำหรับทุกคนในขณะที่สนับสนุนผู้ที่ต้องการพาการบันทึกฟิล์มของตนไปไกลกว่านี้

รุ่นนี้ยังรวมถึงการปรับปรุงประสิทธิภาพทั่วทั้งแอป องค์ประกอบการออกแบบที่ปรับแต่งเพื่อความชัดเจนที่ดีขึ้น และความเสถียรที่เพิ่มขึ้นเมื่อจัดการคลังฟิล์มขนาดใหญ่ ไอคอนแอปและหน้าจอเริ่มต้นที่อัปเดตสะท้อนถึงการพัฒนาของ Frames ในฐานะเครื่องมือมืออาชีพสำหรับช่างภาพฟิล์ม

  • บริการสมาชิก Frames Pro: ปลดล็อกม้วนฟิล์มไม่จำกัดและเข้าถึงฟีเจอร์ขั้นสูงก่อนใครเมื่อมีการเปิดตัว
  • รักษาแพ็กเกจฟรี: ใช้ฟีเจอร์หลักทั้งหมดต่อได้สูงสุด 10 ม้วนฟิล์มโดยไม่มีค่าใช้จ่าย
  • ปรับปรุงประสิทธิภาพ: การตอบสนองและความเสถียรที่เพิ่มขึ้นทั่วทั้งแอป
  • ปรับแต่งการออกแบบภาพ: อัปเดตไอคอนแอป หน้าจอเริ่มต้น และองค์ประกอบอินเทอร์เฟซเพื่อประสบการณ์ที่ลงตัว
  • แก้ไขบั๊กการส่งออก: แก้ไขปัญหาที่ไฟล์ส่งออกไม่สร้างใหม่อย่างถูกต้องหลังการเปลี่ยนแปลง

การปรับปรุง (7)

  • เปิดตัวบริการสมาชิก Frames Pro พร้อมม้วนฟิล์มไม่จำกัด
  • เพิ่มประสิทธิภาพและการตอบสนองของแอปโดยรวม
  • ปรับแต่งการออกแบบ user interface ทั่วทั้งแอป
  • อัปเดตไอคอนแอปด้วยเอกลักษณ์ภาพที่ปรับแต่งแล้ว
  • ออกแบบหน้าจอเริ่มต้นใหม่เพื่อความประทับใจแรกที่ดีขึ้น
  • ปรับปรุงการใช้หน่วยความจำสำหรับคลังฟิล์มขนาดใหญ่
  • ปรับปรุงความเสถียรเมื่อเพิ่มอุปกรณ์กล้องและเลนส์

การแก้ไข (1)

  • แก้ไขบั๊กที่ป้องกันไม่ให้ไฟล์ส่งออกสร้างใหม่หลังการเปลี่ยนแปลง

แพตช์ (7)

  • 1.1.1: อัปเดตไอคอนแอปเล็กน้อย
  • 1.1.2: ปรับปรุงประสิทธิภาพเล็กน้อย
  • 1.1.3: ไอคอนแอปใหม่
  • 1.1.4: ออกแบบไอคอนแอปใหม่
  • 1.1.5: อัปเดตหน้าจอเริ่มต้นของแอป
  • 1.1.6: แก้ไขบั๊กที่ป้องกันไม่ให้ไฟล์ส่งออกสร้างใหม่อย่างถูกต้องหลังการเปลี่ยนแปลง
  • 1.1.7: ปรับปรุงความเสถียรเมื่อเพิ่มอุปกรณ์บน iOS 26 Beta
macOS 1.008 เม.ย. 2025

เปิดตัว Frames สำหรับ macOS

แอป macOS ที่รอคอยมานานเปิดตัวแล้ว นำมาซึ่งการรวม metadata กลับเข้าไปในภาพฟิล์มที่สแกนได้อย่างราบรื่น นำเข้าข้อมูลม้วนฟิล์มของคุณจาก iOS และฝัง EXIF metadata เข้าไปในไฟล์ JPEG

การรวม Metadata สำหรับฟิล์มที่สแกน

Frames สำหรับ macOS แก้ปัญหาฟีเจอร์ที่ชุมชนช่างภาพฟิล์มขอมากที่สุด: การนำ metadata กลับเข้าไปในภาพที่สแกน นำเข้าไฟล์ .frames ของคุณจาก iOS ลากภาพ JPEG ที่สแกนเข้ามา และแอปจับคู่เฟรมกับภาพ การตั้งค่าแสง ข้อมูลกล้อง ข้อมูลเลนส์ ฟิล์ม และบันทึกการถ่ายของคุณถูกฝังเป็น EXIF metadata ที่ทำงานได้อย่างราบรื่นกับ Lightroom, Capture One และ Apple Photos

ระบบการทำงานที่เรียบง่าย ผลลัพธ์มืออาชีพ

ระบบการทำงานตรงไปตรงมา: นำเข้าไฟล์ .frames ของคุณ เพิ่มภาพที่สแกน จับคู่เฟรมกับภาพ และส่งออก แอปจัดการการฝัง EXIF โดยอัตโนมัติ ทำให้เอกสารการถ่ายภาพ analog ของคุณกลายเป็นบันทึกดิจิทัลถาวร ข้อมูลม้วนฟิล์มของคุณถ่ายโอนได้อย่างราบรื่นจากการบันทึกในสนามบน iOS ไปยังการเก็บถาวรขั้นสุดท้ายบน macOS

การปรับปรุง (6)

  • นำเข้าไฟล์ .frames จาก iOS
  • รองรับไฟล์ JPEG พร้อมการฝัง metadata เต็มรูปแบบ
  • การจับคู่เฟรมกับภาพอัตโนมัติ
  • การฝัง EXIF metadata เข้ากันได้กับ Lightroom, Capture One และ Apple Photos
  • การออกแบบ macOS แท้ตาม Human Interface Guidelines ของ Apple
  • มีให้บริการใน 7 ภาษา: อังกฤษ ฝรั่งเศส สเปน อิตาลี โปรตุเกส เยอรมัน และดัตช์

แพตช์ (7)

  • 1.0.1: เพิ่มการรองรับภาษาฝรั่งเศส
  • 1.0.2: ขยายเป็น 7 ภาษา: อังกฤษ ฝรั่งเศส สเปน อิตาลี โปรตุเกส เยอรมัน และดัตช์
  • 1.0.3: ปรับปรุงการรองรับภาษา
  • 1.0.4: ตอนนี้เข้ากันได้กับ macOS 15.0 และใหม่กว่า
  • 1.0.5: ภาพที่ส่งออกตอนนี้เก็บชื่อไฟล์ต้นฉบับ
  • 1.0.6: ออกแบบไอคอนแอปใหม่
  • 1.0.7: ปรับแต่งโลโก้แอปเล็กน้อย
iOS 1.001 เม.ย. 2025

เปิดตัว Frames: สมุดบันทึกถ่ายภาพฟิล์มของคุณ

Frames เปิดตัวในฐานะการสร้างใหม่ทั้งหมดของ Datafilm นำเสนอแอป iOS ที่ปรับปรุงใหม่สำหรับบันทึก metadata การถ่ายภาพฟิล์มในสนาม พร้อมด้วยแอป macOS ใหม่สำหรับฝังข้อมูลนั้นเข้าไปในภาพที่สแกนของคุณ

บันทึกและจัดการม้วนฟิล์มของคุณ

Frames คือสมุดบันทึกดิจิทัลสำหรับการถ่ายภาพแบบ analog ที่ออกแบบมาเพื่อบันทึกการตั้งค่าการถ่าย และผสานรวม metadata เข้าไปในภาพที่สแกนได้อย่างราบรื่น บันทึกรายละเอียดทุกอย่างที่สำคัญสำหรับแต่ละเฟรม: ความยาวโฟกัส ความเร็วชัตเตอร์ ค่าชดเชยแสง รูรับแสง การตั้งค่าแฟลช กล้อง เลนส์ ฟิล์ม วันที่ เวลา และตำแหน่ง

  • บันทึก Metadata ได้ครบถ้วน: บันทึกหมายเลขเฟรม ความยาวโฟกัส ความเร็วชัตเตอร์ ค่าชดเชยแสง รูรับแสง การตั้งค่าแฟลช กล้อง เลนส์ ฟิล์ม วันที่ เวลา และตำแหน่ง
  • ภาพอ้างอิง: แนบภาพอ้างอิงให้กับแต่ละเฟรมเพื่อดูบริบทภาพได้อย่างรวดเร็วเมื่อจับคู่กับภาพที่สแกน
  • มุมมองแผนที่แบบโต้ตอบ: ดูภาพฟิล์มของคุณบนแผนที่แบบโต้ตอบเพื่อย้อนกลับไปยังตำแหน่งที่ถ่ายแต่ละเฟรม
  • การจัดระเบียบด้วยโฟลเดอร์: จัดระเบียบคลังฟิล์มของคุณด้วยโฟลเดอร์และเครื่องมือค้นหาที่ทรงพลังเพื่อค้นหาเฟรมใดๆ ได้ทันที

ส่งออกและซิงค์ข้อมูลของคุณ

แอป macOS ที่ทำงานคู่กันช่วยแก้ปัญหาฟีเจอร์ที่ชุมชนช่างภาพฟิล์มขอมากที่สุด: การนำ metadata กลับเข้าไปในภาพ JPEG ที่สแกนแล้ว ส่งออก metadata ของคุณเป็นไฟล์ JSON (.frames) แชร์บันทึกระหว่างอุปกรณ์ Apple และใช้แอป macOS เพื่อฝัง EXIF data ที่ทำงานได้อย่างราบรื่นกับ Lightroom, Capture One และ Apple Photos

ปรับให้เหมาะสมสำหรับการถ่ายภาพแบบ Analog

  • ตัวควบคุมที่ปรับตัวได้: ตัวควบคุมปรับตัวตามลักษณะของอุปกรณ์ของคุณเพื่อการบันทึกที่รวดเร็วขึ้น
  • ระบบกรอกอัตโนมัติอัจฉริยะ: ระบบกรอกอัตโนมัติอัจฉริยะสำหรับเลนส์ กล้อง ฟิล์ม และ ISO ช่วยประหยัดเวลาในสนาม
  • ฟิล์มมากกว่า 200 ชนิด: มีฟิล์มยอดนิยมจากผู้ผลิตหลักๆ โหลดไว้ให้แล้วมากกว่า 200 ชนิด
  • การออกแบบของ Apple แท้: ปฏิบัติตาม Human Interface Guidelines ของ Apple เพื่อประสบการณ์ที่ราบรื่นและใช้งานง่าย
  • ปรับประสิทธิภาพให้เหมาะสม: ทำงานได้อย่างราบรื่นแม้บน iPhone รุ่นเก่า

การปรับปรุง (12)

  • สร้างใหม่ทั้งหมดตั้งแต่ต้นด้วยสถาปัตยกรรมที่ปรับปรุงแล้ว
  • บันทึก metadata ของเฟรมอย่างครบถ้วน: แสง กล้อง เลนส์ ฟิล์ม ตำแหน่ง
  • บันทึกเวลาอัตโนมัติสำหรับทุกเฟรม
  • แนบภาพอ้างอิงสำหรับการจับคู่กับภาพที่สแกน
  • การจัดระเบียบด้วยโฟลเดอร์สำหรับคลังม้วนฟิล์ม
  • การค้นหาที่ทรงพลังในทุกเฟรมที่บันทึกไว้
  • มุมมองแผนที่แบบโต้ตอบแสดงตำแหน่งการถ่าย
  • แอป macOS ที่ทำงานคู่กันสำหรับฝัง EXIF metadata
  • เข้ากันได้เต็มรูปแบบกับ Lightroom, Capture One และ Apple Photos
  • การออกแบบที่เน้นความเป็นส่วนตัวโดยไม่ต้องมีบัญชีหรือพึ่งพาระบบคลาวด์
  • ปรับปรุง user interface ของแถบเครื่องมือ
  • เพิ่มการรองรับหลายภาษารวมถึงฝรั่งเศส เยอรมัน สเปน อิตาลี ดัตช์ โปรตุเกส โปแลนด์ และสวีเดน

การแก้ไข (2)

  • แก้ไขปัญหาที่การเพิ่มกล้องหรือเลนส์ใหม่ทำให้หน้าจอค้างในบางอุปกรณ์
  • แก้ไข URL ของแอปรีวิว

แพตช์ (8)

  • 1.0.1: ปรับปรุงเล็กน้อยและแก้ไขบั๊ก
  • 1.0.2: แก้ไขเล็กน้อยและปรับปรุงการแปลภาษา
  • 1.0.3: ปรับปรุงการแปลภาษา
  • 1.0.4: ปรับปรุง UI แถบเครื่องมือและขยายการแปลภาษา
  • 1.0.5: เพิ่มการแปลภาษา
  • 1.0.6: แก้ไขปัญหาที่การเพิ่มกล้องหรือเลนส์ใหม่ทำให้หน้าจอค้างในบางอุปกรณ์
  • 1.0.7: แก้ไข URL ของแอปรีวิว
  • 1.0.8: เพิ่มความเข้ากันได้กับ iOS 18.0 และใหม่กว่า
