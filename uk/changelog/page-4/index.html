Підтримка TIFF файлів

Тепер ви можете імпортувати та експортувати TIFF файли з повною інтеграцією метаданих, розширюючи підтримку форматів за межі JPEG для професійних робочих процесів.

Підтримка TIFF прибуває в Frames для macOS. Імпортуйте TIFF скани та вбудовуйте метадані ваших плівкових роликів з тим самим робочим процесом, який ви використовуєте для JPEG файлів. TIFF файли безшовно працюють у процесі реінтеграції метаданих, гарантуючи, що ваші нотатки аналогової фотографії подорожують з вашими зображеннями незалежно від формату.

Покращення (3)
  • Додано підтримку імпорту TIFF файлів
  • Додано експорт TIFF файлів з повною інтеграцією метаданих
  • TIFF файли безшовно працюють з існуючим робочим процесом реінтеграції метаданих

Патчі (1)
  • 1.1.1: Виправлено проблему, яка могла запобігти завантаженню .frames файлів у певних ситуаціях.
iOS 1.1 03 чер 2025

Представляємо Frames Pro

Frames Pro доступний, пропонуючи необмежену кількість плівкових роликів і доступ до розширених функцій у міру їх випуску. Безкоштовні користувачі можуть продовжувати користуватися Frames з до 10 роликами, тоді як Pro розкриває повний потенціал вашого робочого процесу аналогової фотографії.

Відкрийте необмежену реєстрацію плівок

З Frames Pro тепер ви можете створювати необмежену кількість плівкових роликів для відстеження всієї вашої колекції аналогової фотографії без обмежень. Незалежно від того, чи документуєте ви сотні роликів з років зйомок, чи активно ведете десятки поточних проектів, Frames Pro масштабується під ваші потреби. Безкоштовні користувачі зберігають повний доступ до всіх основних функцій з обмеженням у 10 роликів, роблячи Frames доступним для всіх, одночасно підтримуючи тих, хто хоче піти далі у веденні записів про плівки.

Цей випуск також включає оптимізацію продуктивності по всьому додатку, вдосконалені елементи дизайну для покращеної чіткості та підвищену стабільність при керуванні великими плівковими бібліотеками. Оновлена іконка додатку та заставка відображають еволюцію Frames як професійного інструменту для плівкових фотографів.

  • Підписка Frames Pro: Відкрийте необмежену кількість плівкових роликів і отримайте ранній доступ до розширених функцій у міру їх випуску.
  • Збережено безкоштовний рівень: Продовжуйте використовувати всі основні функції з до 10 плівковими роликами безкоштовно.
  • Покращення продуктивності: Підвищена швидкість відгуку та стабільність по всьому додатку.
  • Вдосконалений візуальний дизайн: Оновлена іконка додатку, заставка та елементи інтерфейсу для доглянутого досвіду.
  • Виправлення помилки експорту: Вирішено проблему, коли файли експорту не регенерувалися належним чином після змін.

macOS 1.0 08 кві 2025

Запуск Frames для macOS

Довгоочікуваний супутній додаток для macOS виходить, приносячи безшовну реінтеграцію метаданих у відскановані плівкові зображення. Імпортуйте дані ваших плівкових роликів з iOS і вбудовуйте EXIF метадані у JPEG файли.

Реінтеграція метаданих для відсканованої плівки

Frames для macOS вирішує найбільш запитувану функцію від спільноти плівкової фотографії: реінтеграцію метаданих у відскановані зображення. Імпортуйте ваші .frames файли з iOS, перетягніть відскановані JPEG зображення, і додаток зіставить кадри з фотографіями. Ваші налаштування експозиції, дані камери, інформація про об'єктив, типи плівок і нотатки зйомки вбудовуються як EXIF метадані, які безшовно працюють з Lightroom, Capture One та Apple Photos.

Простий робочий процес, професійні результати

Робочий процес простий: імпортуйте ваш .frames файл, додайте відскановані зображення, зіставте кадри з фотографіями та експортуйте. Додаток автоматично обробляє вбудовування EXIF, гарантуючи, що ваша документація аналогової фотографії стає постійними цифровими записами. Дані ваших плівкових роликів безшовно переносяться з польового логування на iOS до фінального архівування на macOS.

Покращення (6)
  • Імпорт .frames файлів з iOS

  • Автоматичне зіставлення кадрів з фотографіями
  • Вбудовування EXIF метаданих, сумісне з Lightroom, Capture One та Apple Photos
  • Нативний дизайн macOS згідно з рекомендаціями Apple Human Interface Guidelines
  • Доступно 7 мовами: англійська, французька, іспанська, італійська, португальська, німецька та нідерландська

iOS 1.0 01 кві 2025

Запуск Frames: Ваш цифровий блокнот для аналогової фотографії

Frames виходить як повна перебудова Datafilm, пропонуючи вдосконалений додаток для iOS для реєстрації метаданих плівкової фотографії в полі та новий додаток для macOS для вбудовування цих даних у відскановані знімки.

Фіксуйте і організовуйте ваші плівкові ролики

Frames — це ваш цифровий блокнот для аналогової фотографії, розроблений для відстеження параметрів знімків і безшовної інтеграції метаданих у відскановані зображення. Записуйте кожну важливу деталь для кожного кадру: фокусну відстань, витримку, компенсацію експозиції, діафрагму, налаштування спалаху, камеру, об'єктив, тип плівки, дату, час і геолокацію.

  • Повна реєстрація метаданих: Відстежуйте номер кадру, фокусну відстань, витримку, компенсацію експозиції, діафрагму, налаштування спалаху, камеру, об'єктив, тип плівки, дату, час і локацію.
  • Референсні зображення: Додавайте референсні зображення до кожного кадру для швидкого візуального контексту при зіставленні зі сканами.
  • Інтерактивний перегляд на карті: Переглядайте ваші знімки на інтерактивній карті, щоб простежити, де був зроблений кожен кадр.
  • Організація в папках: Організуйте вашу плівкову бібліотеку за допомогою папок і потужних інструментів пошуку для миттєвого знаходження будь-якого кадру.

Експортуйте і синхронізуйте ваші дані

Супутній додаток для macOS вирішує найбільш запитувану функцію від спільноти плівкової фотографії: реінтеграцію метаданих у відскановані JPEG зображення. Експортуйте ваші метадані як JSON (.frames) файли, діліться записами між пристроями Apple та використовуйте додаток для macOS для вбудовування EXIF даних, які безшовно працюють з Lightroom, Capture One та Apple Photos.

Оптимізовано для аналогової фотографії

  • Адаптивні елементи керування: Елементи керування адаптуються до характеристик вашого обладнання для швидшої реєстрації.
  • Розумне автозаповнення: Інтелектуальне автозаповнення для об'єктива, камери, типу плівки та ISO економить час у полі.
  • Понад 200 типів плівок: Попередньо завантажена колекція популярних плівок від провідних виробників.
  • Нативний дизайн Apple: Відповідає рекомендаціям Apple Human Interface Guidelines для плавного, інтуїтивного досвіду.
  • Оптимізована продуктивність: Працює плавно навіть на старших моделях iPhone.
