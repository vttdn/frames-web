Lançamento do Frames: Seu Caderno de Fotografia Analógica

O Frames chega como uma reconstrução completa do Datafilm, trazendo um aplicativo iOS refinado para registrar metadados de fotografia analógica em campo, combinado com um novo aplicativo macOS para incorporar esses dados em suas imagens digitalizadas.

Capture e Organize Seus Filmes

O Frames é seu caderno digital para fotografia analógica, projetado para registrar configurações de disparo e integrar perfeitamente metadados em imagens digitalizadas. Registre cada detalhe importante de cada fotograma: distância focal, velocidade do obturador, compensação de exposição, abertura, configurações de flash, câmera, lente, película, data, hora e geolocalização.

  • Registro Completo de Metadados: Registre número do fotograma, distância focal, velocidade do obturador, compensação de exposição, abertura, flash, câmera, lente, película, data, hora e localização.
  • Imagens de Referência: Anexe imagens de referência a cada fotograma para contexto visual rápido ao combinar digitalizações.
  • Visualização de Mapa Interativo: Visualize seus disparos em um mapa interativo para refazer o caminho de onde cada fotograma foi capturado.
  • Organização por Pastas: Organize sua biblioteca de filmes com pastas e ferramentas de busca poderosas para encontrar qualquer fotograma instantaneamente.

Exporte e Sincronize Seus Dados

O aplicativo complementar para macOS resolve o recurso mais solicitado pela comunidade de fotografia analógica: reintegrar metadados em imagens JPEG digitalizadas. Exporte seus metadados como arquivos JSON (.frames), compartilhe registros entre dispositivos Apple e use o aplicativo macOS para incorporar dados EXIF que funcionam perfeitamente com Lightroom, Capture One e Fotos da Apple.

Otimizado para Fotografia Analógica

  • Controles Adaptativos: Os controles se adaptam às características do seu equipamento para registro mais rápido.
  • Preenchimento Automático Inteligente: Preenchimento automático inteligente para lente, câmera, película e ISO economiza tempo em campo.
  • Mais de 200 Películas: Seleção pré-carregada de películas populares dos principais fabricantes.
  • Design Nativo Apple: Segue as Diretrizes de Interface Humana da Apple para uma experiência suave e intuitiva.
  • Desempenho Otimizado: Funciona perfeitamente mesmo em modelos iPhone mais antigos.
