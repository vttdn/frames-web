Funktioner

Arkiv

Organisera Bibliotek

Bläddra, organisera och hantera alla dina filmrullar enkelt, håll varje session tydligt strukturerad, lätt att hitta och perfekt organiserad.

Utrustning

Hantera Utrustning

Håll en detaljerad förteckning över dina kameror, objektiv och filter för att vara konsekvent och ha all utrustningsinfo redo för varje fotografering.

Spela in

Fånga Inställningar

Logga bländare, slutartid, filmtyp och platsdetaljer direkt för varje bild, håll varje bilds information korrekt och tillgänglig.

Historik

Återbesök Bilder

Titta tillbaka på dina loggade rullar för att granska inställningar, analysera tidigare sessioner, lär av dem och förfina ditt filmfotografiarbetsflöde.

Export

Exportera Poster

Spara eller dela dina poster som .frames, .txt eller .gpx-filer för säkerhetskopiering, delning eller integrering med andra appar eller arbetsflöden.

Metadata

Spara Metadata

Bädda in all inspelad data i skannade bilder för att bevara analog information samtidigt som dina digitala arkiv hålls helt synkroniserade.

Support

E-postsupport

Kontakta utvecklaren direkt för hjälp, dela feedback eller föreslå nya funktioner och förbättringar när som helst, med snabba svar.

Frågor & Svar

Vilka metadata kan jag registrera?

Frames låter dig logga detaljerad information för varje bild, inklusive kamera, objektiv, filter, bildnummer, brännvidd, bländare, slutartid, ISO, mätläge, filmtyp, blixt, GPS-plats, tidsstämplar och egna anteckningar, så du aldrig förlorar viktiga analogfotografidetaljer.

Vilka bildformat stöds?

Frames stöder JPEG, TIFF och DNG-format, vilket låter dig bädda in dina registrerade metadata direkt i skannade analoga foton för sömlös organisation och arbetsflöde mellan dina enheter.

Finns Frames på Android eller Windows?

Frames är för närvarande tillgänglig för iPhone och macOS. Versioner för Android och Windows planeras till 2026, och under tiden kan du dela och hantera din data via .frames-filexport.

Vilka slutartider och bländare stöds?

Frames erbjuder standardsekvenser som täcker de flesta objektiv och kameror, och du kan även definiera egna sekvenser för din utrustning, vilket gör den kompatibel med alla slutartids- eller bländarvärden.

Kan jag logga stativ eller tillbehör?

Ja, Frames låter dig registrera stativ, blixtar och andra tillbehör. Det finns även ett extra anteckningsfält där du kan lägga till egna detaljer du vill.

Vad finns det för planer för framtida uppdateringar?

Framtida planer inkluderar utökade arbetsflödesfunktioner, förbättrad integration mellan enheter och nya verktyg som efterfrågats av gemenskapen för att göra analogfotografiloggning ännu mer flexibel och kraftfull.
