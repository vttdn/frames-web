Introduzione

In Frames, la tua privacy è la nostra priorità. La presente Informativa sulla privacy spiega come gestiamo le informazioni all’interno delle nostre app per iOS e macOS e sul nostro sito web. Ci impegniamo a mantenere i tuoi dati sicuri, in modo trasparente e sempre sotto il tuo controllo. Non effettuiamo alcuna raccolta o tracciamento non necessario — sei sempre tu a decidere quali informazioni condividere o esportare.

1. Frames per iOS e macOS

Frames non raccoglie, archivia né condivide dati personali.
Tutte le informazioni inserite nell’app vengono salvate localmente sul tuo dispositivo e non vengono mai trasmesse a server o terze parti.

L’app utilizza il framework StoreKit di Apple per la gestione degli abbonamenti. Questo processo è interamente gestito da Apple e nessuna informazione personale è accessibile allo sviluppatore.

Frames non include sistemi di analisi, tracciamento o servizi online. Non stabilisce connessioni di rete, ad eccezione dell’uso del servizio di localizzazione del sistema (CLLocationManager) per registrare le coordinate GPS, se decidi di farlo.

Durante l’esportazione dei dati, l’app può creare file .frames, .txt o .gpx. La condivisione di questi file tramite il menu di condivisione del sistema è completamente sotto il tuo controllo. Lo sviluppatore non ha accesso né effettua alcun tipo di tracciamento sui contenuti esportati.

Frames richiede le seguenti autorizzazioni di sistema:

  • Fotocamera: per scattare una foto direttamente dall’app.
  • Foto: per selezionare immagini di riferimento dalla tua libreria.
  • Posizione (Durante l’uso dell’app): per registrare le coordinate GPS quando acquisisci i tuoi fotogrammi.

2. Sito web withframes.com

Il sito web di Frames è un sito statico ospitato su GitHub Pages.
Non utilizza cookie, strumenti di analisi o script di tracciamento. Nessun dato personale viene raccolto o archiviato.

Cloudflare fornisce statistiche di traffico di base e anonime basate sui log del server. Questi log non contengono informazioni personali e non vengono utilizzati per profilazione o marketing.

Il sito include collegamenti a siti web esterni e piattaforme social per la condivisione di contenuti o la citazione di pubblicazioni. La visita di tali siti è facoltativa ed è soggetta alle rispettive informative sulla privacy.

Questo sito web non raccoglie informazioni tramite moduli, newsletter o altre funzionalità interattive.

3. Contatti

Per domande o dubbi riguardanti la presente informativa sulla privacy, puoi contattare lo sviluppatore al seguente indirizzo: hi@withframes.com.
