Обновления

iOS 1.605 сен 2025·Обновления

Пользовательские последовательности выдержек и улучшения карты

Устанавливайте пользовательские последовательности выдержек для каждой камеры, показывая только те скорости, которые фактически поддерживает ваша камера. Это обновление также приносит улучшенную обработку карты, дополнительные опции выдержек и индикаторы прогресса экспорта.

Пользовательское управление выдержкой

Настройте, какие выдержки появляются в ползунке рекордера и настройках кадра для каждой камеры отдельно. Если у вашей камеры есть только выдержки от 1/1000 до 1 секунды, вы можете скрыть опции расширенного диапазона. Это сохраняет интерфейс рекордера чистым и предотвращает случайную запись выдержек, которые ваша камера не поддерживает. Пользовательские последовательности синхронизируются с приложением для Mac для согласованных метаданных на всех устройствах.

Улучшенные функции карты и экспорта

Просмотр карты теперь более надёжно обрабатывает координаты с улучшенной поддержкой жестов для панорамирования и масштабирования. Мы добавили часто используемые выдержки (1/160, 1/80, 1/40, 1/20), которые ранее отсутствовали. Операции экспорта теперь показывают индикатор прогресса, чтобы вы знали, когда ваши файлы генерируются перед появлением листа общего доступа.

Улучшения (5)

  • Добавлены пользовательские последовательности выдержек для каждой камеры
  • Улучшена обработка координат карты и поддержка жестов
  • Добавлены отсутствующие выдержки: 1/160, 1/80, 1/40 и 1/20
  • Добавлен индикатор прогресса экспорта перед появлением листа общего доступа
  • Повышена точность ползунков в виде рекордера

Исправления (1)

  • Точность ползунка рекордера: Исправлены ползунки рекордера для соответствия как правильному значению, так и визуальной позиции.
iOS 1.503 сен 2025

Новые плёнки, переработанная система диафрагмы и переключение объективов

Это обновление расширяет библиотеку плёнок новыми эмульсиями, полностью переработывает управление диафрагмой с пользовательскими последовательностями f-stop и добавляет возможность переключать объективы прямо из рекордера.

Расширенная библиотека плёнок

Список плёнок теперь включает Harman Phoenix II, Lucky Color 200, Leica Monopan 50 и несколько других популярных эмульсий. Снимаете ли вы экспериментальные цветные плёнки или классические чёрно-белые, Frames поддерживает более широкий спектр типов плёнок для более точной записи.

Переработанное управление диафрагмой

Управление диафрагмой было полностью перестроено. Теперь вы можете определять пользовательские последовательности f-stop для каждого объектива, показывая только те значения диафрагмы, которые фактически есть у вашего объектива. Ползунок рекордера теперь правильно центрирован на выбранном значении диафрагмы. Переключайте объективы прямо из рекордера, не прерывая процесс работы, и устанавливайте фокусное расстояние через кнопку «Ещё» для зум-объективов или регулировок перспективного контроля.

Улучшения (7)

  • Расширена библиотека плёнок с Harman Phoenix II, Lucky Color 200, Leica Monopan 50 и другими
  • Полностью переработана система управления диафрагмой
  • Добавлены пользовательские последовательности f-stop для каждого объектива
  • Добавлено переключение объективов прямо из рекордера
  • Фокусное расстояние теперь можно настроить через кнопку «Ещё» в рекордере
  • Улучшена согласованность локализации
  • Незначительные улучшения производительности по всему приложению

Исправления (1)

  • Центрирование ползунка диафрагмы: Исправлена проблема, при которой ползунок диафрагмы в рекордере не был правильно центрирован на выбранном значении.
macOS 1.301 сен 2025

Поддержка DNG Raw-файлов

Теперь вы можете импортировать DNG-файлы наряду с JPEG и TIFF и реинтегрировать метаданные напрямую в ваши raw-изображения для полного процесса аналоговой фотографии.

Поддержка raw-изображений приходит в Frames для macOS. Импортируйте DNG-файлы наряду с вашими существующими процессами JPEG и TIFF и встраивайте метаданные вашей плёнки напрямую в raw-файлы. Это обеспечивает, что ваши настройки экспозиции, данные камеры, плёнки и заметки путешествуют с вашими raw-изображениями через постобработку и архивирование. Мы также исправили проблему, при которой дополнительные заметки для кадра не сохранялись во время реинтеграции метаданных.

Улучшения (3)

  • Добавлена поддержка DNG raw-файлов
  • Полная реинтеграция метаданных для raw-изображений
  • DNG-файлы бесшовно работают наряду с JPEG и TIFF

Исправления (1)

  • Сохранение заметок о кадрах: Исправлена проблема, при которой дополнительные заметки для кадра не сохранялись во время реинтеграции метаданных.
macOS 1.224 авг 2025

Поддержка фильтров, японский и корейский языки

Фильтры теперь появляются на боковой панели и автоматически добавляются в метаданные при сохранении кадров. Это обновление также приносит поддержку японского и корейского языков, плюс важные исправления ошибок.

Фильтры, записанные на iOS, теперь отображаются на боковой панели macOS и автоматически встраиваются в EXIF-метаданные во время реинтеграции. Это завершает вашу аналоговую документацию, включая UV, ND, CPL и другие фильтры в ваших финальных изображениях. Серийные номера камер теперь включены в метаданные, и мы исправили ошибку, при которой значения вспышки были инвертированы. Frames теперь доступен на японском и корейском языках, соответствуя расширению языков iOS.

Улучшения (5)

  • Добавлено отображение фильтров на боковой панели
  • Фильтры автоматически добавляются в метаданные при сохранении кадров
  • Серийный номер камеры теперь включён в EXIF-метаданные
  • Добавлена поддержка японского языка
  • Добавлена поддержка корейского языка

Исправления (1)

  • Инверсия значения вспышки: Исправлена проблема, при которой значение вспышки было инвертировано (показывало «Да», когда должно было быть «Нет»).

Патчи (2)

  • 1.2.2: Улучшена локализация для новых языков.
  • 1.2.1: Улучшены переводы для согласованности с приложением iOS, плюс незначительные улучшения интерфейса.
iOS 1.420 авг 2025

Поддержка записи фильтров

Документируйте свой полный аналоговый процесс, записывая фильтры вместе с камерой, объективом и плёнкой. Теперь вы можете записывать UV, ND, CPL и другие фильтры, используемые для каждого кадра.

Фильтры теперь часть вашего процесса записи плёнки. Записывайте, какие фильтры вы использовали для каждого кадра, включая UV, ND, CPL и другие. Это завершает вашу аналоговую документацию, фиксируя каждый элемент, влияющий на финальное изображение, от корпуса камеры и объектива до плёнки и фильтров. Данные о фильтрах включены во все экспорты и синхронизируются с приложением для Mac для комплексного ведения записей.

Улучшения (5)

  • Добавлена поддержка записи фильтров (UV, ND, CPL и другие)
  • Фильтры включены во все форматы экспорта
  • Данные о фильтрах синхронизируются с приложением для Mac
  • Незначительные улучшения производительности
  • Общие улучшения стабильности

Патчи (2)

  • 1.4.1: Исправлена проблема, при которой редактирование плёнки в подпапке перемещало её обратно в корневую папку.
  • 1.4.2: Улучшены переводы для согласованности с приложением для Mac. Исправлено отображение страны при поиске местоположения. Добавлены значения выдержки и диафрагмы по умолчанию при создании плёнок.
